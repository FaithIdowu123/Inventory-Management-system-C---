# Overview

This project is a simple inventory management system written in C++. The program allows a user to manage a collection of items by storing information such as the item name, price, and quantity. Through a menu-driven interface in the terminal, users can add items, view the current inventory, update item details, remove items, search for items, and save or load inventory data from a CSV file.

The purpose of this software is to strengthen my understanding of C++ programming and object-oriented design. While developing this program, I focused on learning how to organize code using classes and functions, manage data using containers, and work with file input and output. This project also helped me practice handling user input and structuring a program so that it is easier to maintain and expand.

This project demonstrates important C++ concepts such as classes, lists from the Standard Template Library, file streams, and modular program design. Building this program helped me better understand how data can be stored, modified, and persisted using files.

# Development Environment

The software was developed using **Visual Studio** as the primary development environment. The program was written and tested using a standard C++ compiler.

The project was written in the **C++ programming language** and makes use of the **Standard Template Library (STL)**. In particular, the program uses the `list` container to store inventory items dynamically. The project also uses standard C++ libraries for file input and output, string handling, and user input.
